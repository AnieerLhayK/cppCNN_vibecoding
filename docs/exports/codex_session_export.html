项目目标

完成一个“基于纯 C++ 的 CNN 交通标志识别系统”。

要求程序能够读取一个图片集合，使用 CNN 对交通标志图片进行分类识别，并提供一个简单前端界面用于选择图片、显示图片和输出识别结果。

技术约束

  • 使用 C++ 作为实现语言。
  • 使用 Visual Studio 可编译的工程结构。
  • CNN 的核心结构用 C++ 自己实现，包括：
  • Tensor / Matrix 数据结构
  • 卷积层 Conv
  • ReLU
  • MaxPooling
  • Flatten
  • Fully Connected
  • Softmax
  • Loss / Accuracy 统计
  • 可以使用 OpenCV 做图片读取、缩放、颜色转换、界面显示等基础图像处理，但不要直接调用深度学习框架完成 CNN。
  • 不使用 PyTorch / TensorFlow / Keras 作为最终程序依赖。
  • 可以参考 GitHub 项目 arneuro/cppCNN 的组织方式和 CNN 实现思路，但不要直接照搬；需要形成适合本项目的清晰结构。
  • 使用 Git 管理全过程，每完成一个稳定阶段就提交一次 commit。

模型权重是什么

源代码定义了 CNN 的结构，而模型权重是训练过程中学到的数值参数，包括：

  • 卷积层的卷积核
  • 卷积层偏置
  • 全连接层权重矩阵
  • 全连接层偏置

可以简单理解为：

源代码 = 大脑结构
模型权重 = 学习后形成的记忆

只有结构而没有训练权重，网络只能随机猜测。程序启动时读取 .bin 权重，才具备当前约 89.63% 的识别能力。

权重属于生成产物，因此保存在本地和教师 Release 包中，但不进入 Git。
