# cppCNN Vibe Coding

本仓库用于探索“使用纯 C++ 实现卷积神经网络”的不同 vibe coding 路径。仓库中的各个一级目录代表不同平台、模型或协作方式完成的独立实现，例如：

- `codex/`：由 Codex 协作完成的 C++ CNN 交通标志识别系统。
- `claude/`：预留给 Claude 或对应工作流的独立实现。
- 其他目录：可用于新增不同工具、模型或开发方式的实现。

各实现可以采用不同的工程组织、网络结构和交互方案，但共同遵循以下方向：

- CNN 核心计算使用 C++ 实现，不直接依赖深度学习框架。
- 尽量提供可编译、可运行、可测试的完整项目。
- 每个实现目录自行维护 README、依赖、构建方式、数据集说明和实验记录。
- 大型数据集、训练模型和构建产物不直接提交到仓库。

## 仓库约定

1. 不同实现之间尽量保持目录隔离，避免共享临时构建文件。
2. 临时下载、截图、调试输出和转换文件不进入源码目录。
3. 每个稳定阶段使用清晰的 Git 提交记录。
4. 新增实现时，在本 README 的“当前实现”表格中登记。
