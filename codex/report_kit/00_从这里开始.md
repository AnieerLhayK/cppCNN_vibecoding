# 从这里开始

你不需要会 Git、GitHub、Visual Studio、CMake 或 Qt。报告资料包已经把可运行程序、源代码和报告材料分开整理好了。

## 第一步：完整解压

1. 右键 ZIP 文件。
2. 选择“全部解压缩”。
3. 不要直接在压缩软件窗口中运行程序。
4. 解压后的整个目录不要随意拆散。

## 第二步：运行程序

进入：

```text
application/
```

双击：

```text
run_demo.bat
```

程序打开后，点击界面中的任意演示图片，再点击识别按钮。界面会显示预测类别、置信度、Top-3 结果和推理耗时。

## 第三步：找到报告材料

报告初稿和设计资料位于：

```text
report_materials/
```

建议先阅读：

```text
project_report.md
design_notes.md
dataset_guide.md
```

源代码位于：

```text
source/
```

不要从头到尾逐行阅读。请按照 `02_代码阅读指南.md` 给出的顺序查看关键模块。

## 不要做的事情

- 不要删除 `application/models/`。
- 不要删除 Qt DLL、`plugins/` 或 `qml/`。
- 不要把程序描述为使用 PyTorch 或深度学习框架。
- 不要把当前 10 类模型描述为完整 43 类模型。
- 不要声称 `92.63%` 是当前便携演示模型的准确率；当前演示模型是 `89.63%`。
