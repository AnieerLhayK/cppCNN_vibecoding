# cppCNN Codex 课程报告资料包

本资料包只包含 `codex/` 的纯 C++ CNN 交通标志识别系统，不包含仓库中的 Claude/PyTorch 原型。

第一次使用请按顺序阅读：

1. [`01_宝宝级资料包指南.md`](01_宝宝级资料包指南.md)
2. [`02_代码阅读指南.md`](02_代码阅读指南.md)

## 资料包用途

- `application/`：无需安装开发工具，解压后直接演示。
- `source/`：报告和答辩需要查看的完整 Codex 源码。
- `report_materials/`：项目报告初稿、设计说明、数据集说明、训练说明、界面截图和 AI 使用记录。

## 必须记住

- 当前便携包包含默认 10 类模型和可手动切换的 43 类 Enhanced 模型。
- 默认 10 类模型适合快速演示；43 类模型适合展示完整 GTSRB 训练成果。
- CNN 核心由 C++ 自己实现，没有调用 PyTorch、TensorFlow 或 Keras。
- LibTorch/CUDA 只作为开发训练加速工具，最终 GUI 加载的是 `.bin` 权重。
- 模型和 Qt 运行库放在应用目录中；完整数据集没有放入资料包。
