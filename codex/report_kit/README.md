# cppCNN Codex 课程报告资料包

本资料包只包含 `codex/` 的纯 C++ CNN 交通标志识别系统，不包含仓库中的 Claude/PyTorch 原型。

第一次使用请按顺序阅读：

1. [`00_从这里开始.md`](00_从这里开始.md)
2. [`01_宝宝级运行指南.md`](01_宝宝级运行指南.md)
3. [`02_代码阅读指南.md`](02_代码阅读指南.md)
4. [`03_报告撰写指南.md`](03_报告撰写指南.md)
5. [`04_答辩与演示指南.md`](04_答辩与演示指南.md)

## 资料包用途

- `application/`：无需安装开发工具，解压后直接演示。
- `source/`：报告和答辩需要查看的完整 Codex 源码。
- `report_materials/`：项目报告初稿、设计说明、数据集说明、训练说明和界面截图。

## 必须记住

- 当前便携演示模型识别 10 类 GTSRB 交通标志。
- 当前便携模型测试集 Top-1 准确率为 `89.63%`。
- 系统代码能够配置为 43 类，但本资料包不能被描述成“已经发布 43 类高精度模型”。
- CNN 核心由 C++ 自己实现，没有调用 PyTorch、TensorFlow 或 Keras。
- 模型和 Qt 运行库放在应用目录中；完整数据集没有放入资料包。
