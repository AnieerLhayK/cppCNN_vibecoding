# 宝宝级运行指南

## 正常演示

1. 完整解压资料包。
2. 打开 `application/`。
3. 双击 `run_demo.bat`。
4. 等待主界面出现。
5. 点击下方不同类别的演示图片。
6. 点击 `Recognize`，或按 `Ctrl+Enter`。
7. 观察 Top-1 类别、置信度、Top-3 概率和推理时间。

也可以按 `Ctrl+O` 选择自己的 PPM、PNG、JPEG 或 BMP 图片。

## 演示图片

资料包提供五种不同类别：

- 30 km/h 限速
- 100 km/h 限速
- 禁止超车
- 3.5 吨以上车辆禁止超车
- 下个路口优先通行

这些图片已经逐张验证，可用于课堂演示和截图。

## 快捷键

- `Ctrl+O`：打开图片。
- `Ctrl+Enter`：运行识别。
- `Esc`：清除当前图片。

## 每个重要文件是干什么的

| 文件或目录 | 作用 | 能否删除 |
| --- | --- | --- |
| `cppcnn_gui.exe` | 图形界面程序 | 否 |
| `cppcnn_app.exe` | 命令行训练和推理程序 | 否 |
| `models/gtsrb_subset10.bin` | 已训练 CNN 参数 | 否 |
| `labels.txt` | 输出编号对应的类别名称 | 否 |
| `demo_images/` | 课堂演示图片 | 建议保留 |
| `Qt6*.dll` | Qt 运行库 | 否 |
| `plugins/`、`qml/` | GUI 运行资源 | 否 |

模型权重可以理解为 CNN 训练后学到的大量数值参数。程序结构写在源代码中，模型真正学到的识别能力保存在 `.bin` 文件中。

## 常见问题

### 双击没有反应

确认已经完整解压，而不是在 ZIP 预览窗口中运行。再尝试直接双击 `cppcnn_gui.exe`。

### Windows 显示安全提醒

本项目没有商业代码签名。确认文件来自本仓库的 GitHub Release 后，可选择“更多信息”并继续运行。

### 提示模型缺失

确认以下文件存在：

```text
application/models/gtsrb_subset10.bin
```

不要单独移动 EXE；应保持整个 `application/` 目录结构。

### 图片打不开

优先使用资料包自带的演示图。自选图片建议使用 PNG、JPEG、BMP 或 PPM。

### 识别结果不理想

当前模型只支持固定的 10 个类别。输入其他交通标志时，网络仍会从这 10 类中选择最接近的一类，这不代表它支持所有 43 类。
