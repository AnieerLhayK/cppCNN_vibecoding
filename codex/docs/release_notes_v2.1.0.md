# cppCNN Traffic Sign Studio v2.1.0

This release focuses on packaging and handoff polish rather than a model change.

## What is included

- `cppCNN-Traffic-Sign-Studio-v2.1.0-windows-x64.zip`
  - Portable Windows runtime package.
  - Includes the Qt GUI, CLI demo, shipped `.bin` models, labels, demo images and AI usage record.
- `cppCNN-Codex-Teacher-Submission-v2.1.0.zip`
  - Teacher-facing submission bundle.
  - Includes the portable application, Codex source tree, compact sample dataset, model metadata and AI records.
- `cppCNN-Codex-User-Software-v2.1.0.zip`
  - End-user software bundle.
  - Includes the runnable application and a `TEST_SET_WEBSITE.url` shortcut to the official GTSRB test-set page.

## Notes

- The full GTSRB dataset and model weights remain excluded from Git history.
- The release assets are intended for direct classroom demonstration and report handoff.
- The user software bundle is the simplest package for classmates who only need to run the demo.
