# Demo Images

这些图片来自本机下载的公开 GTSRB 测试集，仅作为课程演示样例，不属于训练数据提交。样例覆盖多个不同模型类别，避免演示仅体现限速数字识别。

| 文件 | 原始类别 |
| --- | --- |
| `01_speed_limit_30.ppm` | Class 1：30 km/h |
| `02_speed_limit_50.ppm` | Class 2：50 km/h |
| `03_speed_limit_60.ppm` | Class 3：60 km/h |
| `04_speed_limit_80.ppm` | Class 5：80 km/h |
| `05_speed_limit_100.ppm` | Class 7：100 km/h |
| `06_no_passing.ppm` | Class 9：禁止超车 |
| `07_heavy_vehicle_no_passing.ppm` | Class 10：禁止 3.5 吨以上车辆超车 |
| `08_intersection_priority.ppm` | Class 11：下个路口优先通行 |

数据来源和官方下载链接见项目的 `docs/dataset_guide.md`。
