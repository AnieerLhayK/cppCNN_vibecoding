# Demo Images

这些图片来自本机下载的公开 GTSRB 测试集，仅作为课程演示样例，不属于训练数据提交。样例特意覆盖五个不同模型类别，避免演示仅体现限速数字识别。

| 文件 | 原始类别 |
| --- | --- |
| `01_speed_limit_30.ppm` | Class 1：30 km/h |
| `02_speed_limit_100.ppm` | Class 7：100 km/h |
| `03_no_passing.ppm` | Class 9：禁止超车 |
| `04_heavy_vehicle_no_passing.ppm` | Class 10：禁止 3.5 吨以上车辆超车 |
| `05_intersection_priority.ppm` | Class 11：下个路口优先通行 |

数据来源和官方下载链接见项目的 `docs/dataset_guide.md`。
